# 个人 Skills Hub

本仓库存放自定义 Hermes 技能，可导入到其他 Hermes 客户端使用。

## 技能列表

### 1. daily-news（每日新闻）
- **触发**: 用户说"新闻"、"今日新闻"、"新闻搜索"等
- **功能**: 获取每日新闻，按AI科技、国际大事、财经、成都地区分类整理输出
- **使用场景**: 快速获取当日要闻简报

### 2. precious-metals-report（贵金属行情）
- **触发**: 用户说"贵金属"、"贵金属行情"、"金银油"等
- **功能**: 获取每日金银铂钯行情，国内详细+国际简略格式
- **使用场景**: 查询贵金属实时价格

### 3. auto-model-fallback（自动模型降级）
- **触发**: API 返回 5xx 错误、超时、空响应等
- **功能**: 自动识别主模型故障并切换到备用模型（MiniMax-M2.7 → M2.5）
- **使用场景**: 保障服务高可用，主模型失败时自动降级

### 4. skill-pipeline（技能入库流水线）
- **触发**: 用户说"技能入库"、"技能学习"、"初始化技能配置"等
- **功能**: 自动化技能归档、入库和 GitHub 同步流程
- **使用场景**: 将实现的任务流快速封装为可复用技能

### 5. house-price-scraper（房产数据爬虫）
- **触发**: 用户说"二手房行情"、"二手房"、"房情"等
- **功能**: 贝壳网二手房小区均价爬虫，支持城市/小区名/室型筛选，自动按总价排序并计算均价
- **使用场景**: 查询小区二手房行情，了解特定房源总价和单价

---

## 导入其他客户端

复制技能文件夹到目标客户端的 `~/.hermes/skills/research/` 目录即可使用。
