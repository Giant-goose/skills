# precious-metals-report

每日贵金属行情播报技能

## 输出格式

**国内贵金属优先，详细展示：**
- 黄金9999、黄金T+D、黄金现货，白银、铂金、钯金，白银T+D
- 格式：「品名：买入价/卖出价（高XXX 低XXX）」
- 单位：元/克

**国际贵金属简略：**
- 伦敦金、伦敦银、伦敦铂、伦敦钯、COMEX金
- 格式：「品名：买入/卖出 | 品名：买入/卖出 | ...」
- 单位：美元/盎司

**外汇（简略）：**
- 美元汇率，一行带过

## 数据来源

1. 首选：浏览器访问 **https://i.jzj9999.com/quoteh5/**（融通金，数据最全）
2. 若融通金无法访问，用 mcporter 搜索：
   ```
   npx mcporter call minimax.web_search query="黄金白银铂金钯金国际价格 最新行情"
   ```

## 执行步骤

1. 浏览器打开 https://i.jzj9999.com/quoteh5/
2. 读取页面中所有品种的最新价、买价、卖价、高价、低价
3. 按格式输出

国内贵金属（元/克）

黄金T+D：买价/卖价（高XXX 低XXX）
