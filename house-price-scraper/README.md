# 贝壳网成都二手房爬虫

爬取贝壳网成都二手房数据，支持按小区名和室型筛选，自动按总价排序，结果保存为 Excel（含缩略图）。

## 环境要求

- Python 3.10+
- Chrome 浏览器

## 安装

```bash
python -m venv venv
venv\Scripts\activate      # Windows
pip install -r requirements.txt
```

## 用法

```bash
# 命令行参数方式（推荐）
python main.py <小区名> [室型]

# 示例
python main.py 鑫苑鑫都汇 3        # 3室
python main.py 鑫苑鑫都汇 三室二厅  # 中文室型自动识别
python main.py 鑫苑鑫都汇          # 不限室型

# 无参数进入交互模式
python main.py
```

室型支持多种写法：`3` / `3室` / `三室` / `三室二厅` / `3室2厅` 均可识别。

## 输出

数据保存在 `data/{城市}_{小区名}_{室型}/` 目录下：

```
data/
└── 成都_鑫苑鑫都汇_3室/
    ├── beike_成都_鑫苑鑫都汇_3室.xlsx   # 数据表格
    └── imgs/                             # 房源缩略图
        ├── 1.jpg
        ├── 2.jpg
        └── ...
```

每次执行同一小区时，会自动清空旧数据（表格+图片），保证数据最新。

| 字段 | 说明 |
|------|------|
| 小区名称 | 房源所在小区 |
| 室型 | 筛选的室型 |
| 总价 | 房源总价（万元）|
| 单价 | 房源单价（元/平）|
| 简图 | 房源缩略图（嵌入Excel，本地文件）|
| 链接 | 房源详情页链接（可点击跳转）|
| 爬取时间 | 数据爬取时间戳 |
| 均价 | 末行汇总，所有房源单价均值 |

## 错误处理

- 小区名输入错误：检测贝壳网"新上房源通知"弹窗 + 推荐结果与输入无关，自动提示用户检查小区名
- 无对应室型房源：提示暂无该室型房源
- 人机验证：检测到验证页面自动停止，提示重试

## 注意事项

- 当前仅支持成都地区
- 翻页会触发人机验证，默认只爬第1页（约30条）
- 首次运行会自动下载 ChromeDriver，需要网络连接
- 浏览器后台无感知运行（headless模式）
